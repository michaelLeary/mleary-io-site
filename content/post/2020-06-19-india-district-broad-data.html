Task:

  1. Extract table data of India districts from Wikipedia
  2. Combine each table rows into one dataframe;
  3. add a column to combined data frame that provides name of original dataframe - so as to facilitate identifying source of data
  4. reorder the columns in expanded data frame to put original data frame column first.

There are 36 tables of India states and territories listed on the wikipedia page. Each table was extracted and stored in a separate data frame. A comma separated list of individual data frames was put into a vector. And a function then appended a column with the name of individual source data frame to each row of data. After that the individual data frames were bounded into one object and variable columns were reordered.

Difficulty and cumbersome script used to allow for unwanted notes included within population, area, and, density cells. More efficient method is required to remove the commas from character variable to permit easy conversion to numeric form.

The value add in result is to aggregate the total population, area and compute population density by state and territory. And then to provide a complete listing of all data by district for each state.
