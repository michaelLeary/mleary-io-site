Output from this script provides charts of consumption of various food categories (in terms kg/person/year) by country and country groupings. The generated charts use the consumption per person though in the nu3 website that is the source of the data placed emphasis on CO2/person/year.

The script extracts an html table data from a website, tidies the extracted data in R, and, then generates charts that show relationships from the data. The script is virtually all written by Kasia Kulma. The added contribution to Kasiaâ€™s script is converting character variables to factor and numeric. And then short script to generate the following charts.

The data is based on that provided by Food and Agriculture Organization of the United Nations as of 2018. Raw table has been extracted from the table published nu3.
